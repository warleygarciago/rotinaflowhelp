## Como usar esta central

1. Escolha uma categoria no menu lateral.
2. Abra o tutorial desejado.
3. Assista ao vídeo curto.
4. Siga o passo a passo na mesma tela.
5. Consulte erros comuns quando algo não sair como esperado.
