# Como consultar DCTFWeb e gerar DARF

## Resumo
Este tutorial mostra como consultar a DCTFWeb, gerar DARF, ver vencimento, PIX, código de barras e PDF.

## Quando usar
Use após o S-1299 estar aceito.

## Passo a passo
1. Confira se a competência está fechada.
2. Clique em gerar DARF pendentes.
3. Aguarde o retorno.
4. Copie PIX ou código de barras.
5. Baixe o PDF para conferência.

## Erros comuns
- Fechamento eSocial ainda pendente
- DARF gerado mas não exibido de imediato
- Consulta DCTFWeb sem procuração/autorização

## Como resolver
Atualize status, confira certificado/procuração e tente novamente.

## Observações importantes
O PDF fica salvo em storage e pode ser baixado posteriormente.
