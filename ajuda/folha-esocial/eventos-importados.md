# Como conferir eventos importados

## Resumo
Este tutorial mostra como conferir eventos eSocial importados, recibos, retornos e situação dos registros.

## Quando usar
Use após importar um ZIP do eSocial ou quando precisar auditar o histórico técnico.

## Passo a passo
1. Acesse Eventos eSocial.
2. Filtre por aceitos, erros ou folha.
3. Abra o evento desejado.
4. Confira resumo, retorno e XML técnico.
5. Baixe o XML se necessário.

## Erros comuns
- Evento rejeitado confundido com evento substituído
- Histórico com erros antigos
- XML não selecionado para download

## Como resolver
Use a competência fechada e os eventos aceitos como referência principal.

## Observações importantes
Eventos antigos com erro podem existir mesmo quando a competência atual está fechada corretamente.
