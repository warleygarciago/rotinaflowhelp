# Como calcular empresa prestadora

## Resumo
Este tutorial mostra cuidados para apurar empresas prestadoras de serviço. O foco é conferir atividade PGDAS, ISS, Fator R e receitas internas.

## Quando usar
Use para empresas de serviços tributadas nos anexos III ou V.

## Passo a passo
1. Confira se as notas são de serviço.
2. Verifique se há retenções ou particularidades.
3. Selecione a atividade PGDAS correta.
4. Calcule e confira o anexo.
5. Marque como conferida antes de transmitir.

## Erros comuns
- Atividade PGDAS incompatível
- Receita externa informada sem necessidade
- ISS confundido com ICMS

## Como resolver
Reveja a atividade e recalcule antes de transmitir.

## Observações importantes
Para a rotina padrão, use somente valores internos quando não houver receita no exterior.
