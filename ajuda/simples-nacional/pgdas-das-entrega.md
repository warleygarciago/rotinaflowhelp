# Transmitir e gerar DAS

## Para que serve
Enviar a apuração conferida ao PGDAS-D, gerar o DAS e entregar os documentos.

## Passo a passo
1. Confira as notas.
2. Calcule a empresa.
3. Marque a apuração como **Conferida**.
4. Clique em **Transmitir PGDAS conferidas**.
5. Aguarde a transmissão.
6. Clique em **Gerar DAS pendentes**.
7. Confira vencimento, valor, PIX ou código de barras.
8. Entregue os documentos no Nibo.

## Quando não tiver DAS
Se a apuração não gerar valor a pagar, o sistema mostra **Sem DAS**.

## Atenção
Não transmita antes de conferir atividade PGDAS, receita e Fator R.
