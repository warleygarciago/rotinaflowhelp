# Como configurar uma empresa

## Resumo
Este tutorial mostra como preparar uma empresa para apuração do Simples Nacional no Rotina Flow. Use para revisar CNPJ, regime tributário, data de abertura e dados necessários para a rotina fiscal.

## Quando usar
Use ao cadastrar uma nova empresa ou ao revisar uma empresa antes da primeira apuração.

## Passo a passo
1. Acesse o cadastro de empresas.
2. Confira razão social, CNPJ e data de abertura.
3. Selecione o regime Simples Nacional.
4. Informe os dados de integração quando houver.
5. Salve e volte para a tela de Simples Fator R.

## Erros comuns
- CNPJ digitado incorretamente
- Regime tributário diferente de Simples Nacional
- Data de abertura vazia

## Como resolver
Revise os dados cadastrais da empresa e salve novamente. Depois atualize a tela de apuração.

## Observações importantes
A data de abertura ajuda o sistema a tratar empresas novas e proporcionalidade quando aplicável.
