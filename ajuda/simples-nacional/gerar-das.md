# Como gerar ou consultar o DAS

## Resumo
Este tutorial mostra como gerar o DAS após a transmissão PGDAS-D e como consultar vencimento, valor, PIX, código de barras e PDF.

## Quando usar
Use após a apuração estar transmitida no PGDAS-D.

## Passo a passo
1. Confira se PGDAS está transmitido.
2. Clique em gerar DAS pendentes.
3. Aguarde o card atualizar.
4. Copie PIX ou código de barras se necessário.
5. Baixe o PDF para conferência ou envio.

## Erros comuns
- Tentar gerar DAS antes da transmissão
- Competência sem movimento com DAS inexistente
- PDF gerado sem leitura de vencimento

## Como resolver
Atualize a tela, confira a etapa PGDAS e gere novamente se necessário.

## Observações importantes
Quando a empresa está sem movimento, a etapa pode ser concluída sem DAS para manter o fluxo visual correto.
