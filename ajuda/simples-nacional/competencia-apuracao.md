# Como configurar a competência

## Resumo
Este tutorial mostra como selecionar o mês de trabalho para apurar empresas no Simples Nacional. A competência escolhida controla notas, bases, cálculo, PGDAS, DAS e entrega.

## Quando usar
Use sempre que for trabalhar em um mês diferente ou conferir uma competência anterior.

## Passo a passo
1. Abra a tela Simples Fator R.
2. Selecione a competência desejada.
3. Confira se a lista de empresas atualizou.
4. Verifique notas e bases da competência.
5. Inicie o cálculo ou a conferência.

## Erros comuns
- Conferir notas de outro mês
- Calcular com competência errada
- Não atualizar a tela após mudar o mês

## Como resolver
Selecione a competência correta e clique em atualizar antes de continuar.

## Observações importantes
O mês de trabalho fica salvo para facilitar a navegação entre telas.
