# Como cadastrar empresa

## Resumo
Este tutorial mostra como cadastrar uma empresa no Rotina Flow e preencher dados usados por folha, eSocial e fiscal.

## Quando usar
Use ao implantar novo cliente no sistema.

## Passo a passo
1. Acesse Empresas/Implantação.
2. Informe razão social, CNPJ e data de abertura.
3. Preencha dados de folha e eSocial.
4. Informe integrações quando necessário.
5. Salve a empresa.

## Erros comuns
- CNPJ duplicado
- Data de abertura incorreta
- Empresa marcada indevidamente sem folha

## Como resolver
Edite o cadastro e corrija os campos antes de calcular.

## Observações importantes
Dados cadastrais corretos evitam rejeições no eSocial e erros no Simples.
