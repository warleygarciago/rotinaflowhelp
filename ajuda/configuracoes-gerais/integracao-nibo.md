# Como configurar Nibo

## Resumo
Este tutorial mostra onde informar credenciais e identificadores do Nibo usados para documentos, organizações e NFS-e.

## Quando usar
Use antes de enviar documentos ao Nibo ou buscar organizações/NFS-e.

## Passo a passo
1. Acesse Perfil/Certificado.
2. Preencha Accountant ID e User ID quando usados.
3. Informe API Key do Nibo Obrigações.
4. Informe ApiToken Nibo Empresa.
5. Salve e teste a rotina desejada.

## Erros comuns
- Token do módulo errado
- Organization ID ausente na empresa
- Credencial expirada ou inválida

## Como resolver
Confira no Nibo qual módulo gerou a credencial e salve novamente.

## Observações importantes
O Nibo possui módulos separados; credenciais de um módulo podem não funcionar em outro.
