# Como configurar WhatsApp

## Resumo
Este tutorial mostra como configurar o canal WhatsApp para envio de links e documentos aos clientes.

## Quando usar
Use antes de enviar documentos por WhatsApp ou configurar recuperação de senha.

## Passo a passo
1. Acesse Perfil/Certificado.
2. Localize Canal WhatsApp.
3. Informe remetente Twilio.
4. Informe Content SID do template.
5. Salve o canal.

## Erros comuns
- Número fora do formato Twilio
- Template não aprovado
- Canal desativado

## Como resolver
Revise remetente, template e status do canal.

## Observações importantes
Para recuperação de senha, o WhatsApp usado é o cadastrado no perfil do usuário.
