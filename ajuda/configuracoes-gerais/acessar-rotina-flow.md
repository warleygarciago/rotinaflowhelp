# Como acessar o Rotina Flow

## Resumo
Este tutorial mostra como entrar no Rotina Flow, usar login seguro e validar o 2FA quando solicitado.

## Quando usar
Use no primeiro acesso ou quando tiver dificuldade para entrar no sistema.

## Passo a passo
1. Acesse o endereço do Rotina Flow.
2. Informe usuário e senha.
3. Digite o código 2FA se solicitado.
4. Confirme se abriu o painel inicial.
5. Em caso de senha esquecida, use recuperação por WhatsApp cadastrado.

## Erros comuns
- Código 2FA inválido
- Senha esquecida
- WhatsApp de recuperação não cadastrado

## Como resolver
Confira o aplicativo autenticador ou solicite recuperação de senha.

## Observações importantes
O WhatsApp de recuperação precisa estar cadastrado previamente no perfil.
